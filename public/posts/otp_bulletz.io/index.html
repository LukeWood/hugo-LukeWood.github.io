Fault Tolerant Development by Example using OTP

Background

Bulletz.io is a browser game I wrote entirely in Elixir using the phoenix framework. Bulletz.io is written to be highly fault tolerant and concurrent. This was important to me as I wanted to be able to run the game on a cheap server and leave it there long term without having to touch it.

This post will detail how I leveraged OTP to make bulletz.io fault tolerant and highly parallel.
